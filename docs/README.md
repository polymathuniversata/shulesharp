Docs — School Payments (Snippe) FastAPI Module

This docs/ folder contains documentation for the hackathon project: a simple FastAPI backend that generates unique Snippe payment links for parents to pay school fees for a student.

Contents:
- overview.md — project summary and flow
- architecture.md — high-level architecture and components
- setup.md — local setup, env, and run instructions
- api.md — FastAPI endpoints, request/response examples
- snippe_integration.md — how to use Snippe (SDK, links)
- webhooks.md — webhook handling and verification
- error_handling.md — error mapping & troubleshooting
- agent_skill.md — notes on Agent Skill integration

Follow these docs to get the backend running and integrate Snippe for payment links.
